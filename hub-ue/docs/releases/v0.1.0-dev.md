# v0.1.0-dev

Primeira baseline validada do hub Python e do plugin Unreal `QuestSupervisor`.

## Validado

- Hub Python local com FastAPI, WebSocket puro, Pydantic, tópicos e ACK.
- Plugin Unreal conectando como cliente WebSocket do hub.
- Auto-start via `Project Settings > Plugins > Quest Supervisor`.
- Recebimento de comandos em Blueprint com `QuestSupervisorComponent`.
- Actor pronto `QuestSupervisorCommandBridgeActor` para testes sem C++.
- Teste real no Meta Quest 3 usando o projeto externo `TesteVR`.
- Comando `pause-session` retornando `accepted`.
- Comando `unknown-action` retornando `rejected`.

## Compatibilidade conhecida

- Python 3.11 ou superior para o hub.
- Unreal Engine 5.7 no ambiente validado.
- Meta Quest 3 no teste físico validado.
- Protocolo WebSocket JSON em `ws://<host>:8787/ws`.

## Limitações

- Integrações reais de HRV, EEG e smartwatch ainda não estão implementadas.
- O dashboard não faz parte do fluxo ativo; operação atual é por CLI e logs locais.
- Instalação do plugin em projetos consumidores ainda depende de cópia/sincronização local.
- O endpoint do Quest precisa usar o IP LAN da máquina que roda o hub.
